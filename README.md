PS1 Cheat ROM Editor

PS1 Cheat ROM Editor is a WinForms tool for opening, editing, and saving PlayStation cheat cartridge ROMs.

Supports:
	•	Xplorer / Xploder
	•	GameShark
	•	Action Replay
	•	Equalizer

⸻

✨ Features
	•	Edit games, cheats, and code lines
	•	Add and remove entries
	•	Paste multiple codes at once
	•	ROM detection via JSON definitions
	•	FX/Xplorer compressed ROM support
	•	Cheat space usage display
	•	Dump and flash ROMs via Unirom / PSX Serial

⸻

🛠️ Build

Built with:
	•	.NET 8
	•	WinForms

  dotnet publish -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true

🔌 Unirom / PSX Serial

Supports dumping and flashing cartridges using Unirom over a PSX serial cable.

More info: https://unirom.github.io/serial_psx_cable/
